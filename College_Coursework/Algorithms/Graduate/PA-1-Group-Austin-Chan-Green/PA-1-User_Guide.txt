How to run the program?
Double click the PA-1-Executable application.
Clicking a button with number "i" will display Array_i in both unsorted and sorted form.
Clicking the "Save CSV" button will generate a csv file with the array size, value of nlg(n), the time spent in milliseconds and the value of nlg(n) / time.
Clicking the "Quit" button will terminate the program.

How was the spreadsheet generated?
Automatically by the program.
